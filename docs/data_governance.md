# Dataset Governance

## Dataset Source
The dataset used in this project is the PlantVillage dataset obtained from Kaggle:
https://www.kaggle.com/datasets/emmarex/plantdisease

## License
The dataset is publicly available for research and educational purposes.

## Data Privacy
The dataset contains only images of plant leaves and does not include any personally identifiable information (PII). Therefore, there are no privacy concerns.

## Representativeness
The dataset consists of images captured under controlled conditions (clean background, consistent lighting). While this improves model performance, it may limit generalization to real-world environments where lighting and background vary.

## Limitations
- Controlled environment images
- May not represent real-world farm conditions
- Possible bias toward clean, well-captured samples

## Justification
The dataset was selected because:
- It is widely used in plant disease classification research
- It provides labeled data for multiple crops and diseases
- It is suitable for training CNN models within limited time and resources
